#ChosenName

This project renders random greetings,and its using react to render the UI.

## Build With

- Ruby
- Ruby on Rails
- PostgreSQL
- SASS
- CSS
- Bootstrap
- React
- Redux

## Installation
```
  $ git clone git@github.com:SimonGrchevski/Hello.git

  $ cd Hello

  $ bundle

  $ rails db:create

  $ rails db:migrate

  $ rails db:seed

  $ ./bin/webpack-dev-server

  $ rails s
  
```
